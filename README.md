# Undercut

Undercut is a "Data Fusion" application for car pricing and negotiation.

## Tech Stack
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, Shadcn/UI, Framer Motion.
- **Backend:** FastAPI, Pydantic, Gemini API.
- **Scraper:** Go + Colly.
- **Database:** Supabase (PostgreSQL).
- **Vibe Roles:**
  - **The Architect:** UI Components (Atoms).
  - **The UX Engineer:** Visualizations, Motion, & Mobile hooks.
  - **The Integrator:** Page construction & State.
  - **The Controller:** API & Business logic.
  - **The Quant:** FMV Logic & Outlier Math.
  - **The Hunter:** Go Data Ingestion.
  - **The Guardian:** CI/CD & E2E Testing.
